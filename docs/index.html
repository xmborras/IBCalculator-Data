How to update the iOS apps:

  1. Click on any subject below to edit its Grade Boundaries (the minimum % required for each grade).
  2. Changes are saved temporarily as you type.
  3. When finished, click the Export JSON button in the top right.
  4. Replace the docs/ib_data.json file in your repository with the newly downloaded file.
  5. Commit and push to GitHub. All iOS apps will automatically download the new stats on their next launch!
